# 比亚迪半导体销售手册转换项目

## 📖 项目简介

本项目将比亚迪半导体销售情报支持手册（大型Markdown文档）转换为结构化的网页PPT格式，便于销售团队查阅、演示和分享。

**版本**: v1.0 - 第一批内容（产品线情报库：产品1-5）
**完成日期**: 2025-11-20

---

## 🎯 快速开始

### 1️⃣ 查看网页PPT

```bash
# 打开导航页（推荐）
open ppt_output/index.html

# 或直接打开任意产品页面
open ppt_output/01_product-sic.html
```

### 2️⃣ 查看拆分后的Markdown

```bash
# 查看拆分索引
cat split_output/README.md

# 查看具体产品
cat split_output/01_product-sic.md
```

### 3️⃣ 重新生成（如需修改）

```bash
# 1. 拆分源文件
python3 split_markdown.py

# 2. 转换为HTML PPT
python3 convert_to_html_ppt.py
```

## 📞 常见问题

### Q: 如何修改品牌颜色？
编辑 `ppt_output/byd-theme.css` 中的 `:root` 变量。

### Q: 如何添加新产品页面？
1. 在 `split_markdown.py` 添加配置
2. 运行拆分和转换脚本
3. 在 `index.html` 添加导航链接

### Q: 可以导出PDF吗？
可以，在浏览器中打开HTML页面，使用"打印→另存为PDF"。

### Q: 如何整合所有页面为单文件？
参考《网页整合工具-使用说明.md》使用网页整合工具。
