# 比亚迪半导体销售手册转换项目

## 📖 项目简介

本项目将比亚迪半导体销售情报支持手册（大型Markdown文档）转换为结构化的网页PPT格式，便于销售团队查阅、演示和分享。

**版本**: v2.0 - 完整版（全部四个批次）
**完成日期**: 2025-11-20
**覆盖率**: 100%（8,245行全部转换）

---

## 🎯 快速开始

### 1️⃣ 查看网页PPT（推荐）

```bash
# 打开导航页
open ppt_output/index.html
```

浏览器中打开后，您将看到：
- **15个内容页面**的导航卡片
- **6个产品情报**页面
- **6个销售场景**页面
- **2个工具/机制**页面

### 2️⃣ 查看拆分后的Markdown

```bash
# 查看拆分索引
cat split_output/README.md

# 查看具体文件
cat split_output/06_product-mcu.md
cat split_output/07_scenario-01-price-negotiation.md
```

### 3️⃣ 重新生成（如需修改）

```bash
# 1. 拆分第一批（产品1-5）
python3 split_markdown.py

# 2. 拆分第二至第四批（产品6 + 场景 + 工具）
python3 split_markdown_batch2_4.py

# 3. 转换所有markdown为HTML
python3 convert_to_html_ppt.py
```

## 🔮 技术架构

### 转换流程

```
销售手册cladue.md (330KB, 8,245行)
          ↓
    ┌─────────────────┐
    │ 批次1：产品1-5   │ → split_markdown.py
    │ 批次2-4：其余内容│ → split_markdown_batch2_4.py
    └─────────────────┘
          ↓
  15个Markdown文件
          ↓
  convert_to_html_ppt.py
          ↓
  16个HTML文件（含导航）
```

### 技术栈
- **语言**: Python 3.6+（无外部依赖）
- **前端**: HTML5 + CSS3 + JavaScript (ES6)
- **图表**: Mermaid (CDN)
- **样式**: 自定义BYD品牌主题

## 📞 常见问题

### Q: 如何修改品牌颜色？
编辑 `ppt_output/byd-theme.css` 中的 `:root` 变量。

### Q: 如何添加新内容？
1. 在源文件中添加内容
2. 更新拆分脚本的配置
3. 重新运行拆分和转换脚本

### Q: 可以导出PDF吗？
可以，在浏览器中打开HTML页面，使用"打印→另存为PDF"。

### Q: 如何整合所有页面为单文件？
参考《网页整合工具-使用说明.md》使用网页整合工具。
