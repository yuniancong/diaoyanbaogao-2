## 🎯 项目目标

3. **便于整合**：支持使用网页整合工具合并为单页应用

## 📐 技术架构

### 转换流程

```
销售手册cladue.md (330KB, 8245行)
          ↓
    [split_markdown.py]
          ↓
  ┌───────────────────────────┐
  │ 00_overview-intro.md      │
  │ 01_product-sic.md         │
  │ 02_product-igbt.md        │
  │ 03_product-camera.md      │
  │ 04_product-led.md         │
  │ 05_product-current-sensor.md │
  └───────────────────────────┘
          ↓
  [convert_to_html_ppt.py]
          ↓
  ┌───────────────────────────┐
  │ index.html (导航页)        │
  │ 00_overview-intro.html    │
  │ 01_product-sic.html       │
  │ ...                       │
  │ byd-theme.css             │
  └───────────────────────────┘
          ↓
   [网页整合工具] (可选)
          ↓
    merged-pages.html
    (单文件，带标签页切换)
```

## 🐛 已知限制

### 1. Mermaid图表依赖网络
- **问题**: 需要加载CDN资源才能渲染
- **影响**: 离线环境图表无法显示
- **解决**: 可下载Mermaid库到本地并修改引用路径

### 2. 相对路径资源
- **问题**: 如果原markdown包含相对路径的图片，转换后可能失效
- **影响**: 当前手册未使用外部图片，无影响
- **预防**: 未来如需图片，建议使用绝对URL或Data URL

### 3. 复杂Markdown语法
- **问题**: 嵌套列表、复杂表格可能需要手动调整
- **影响**: 当前手册结构简单，未遇到问题
- **建议**: 如遇到格式问题，可手动编辑HTML

## 📞 使用支持

### 常见问题

**Q1: 如何修改品牌颜色？**
A: 编辑 `byd-theme.css` 中的 `:root` 变量：
```css
:root {
  --byd-red: #YOUR_COLOR; /* 修改这里 */
}
```

**Q2: 如何添加新的产品页面？**
A:
1. 在 `split_markdown.py` 添加配置
2. 运行拆分脚本
3. 运行转换脚本
4. 在 `index.html` 添加导航卡片

**Q3: 可以导出为PDF吗？**
A: 可以，在浏览器中打开HTML页面，使用"打印"功能选择"另存为PDF"

**Q4: 如何整合所有页面？**
A: 参考《网页整合工具-使用说明.md》使用网页整合工具
